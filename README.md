# cvs-rm

### canvas小游戏，节奏大师

### 话
> * 使用了Web Worker，需要搭服务器才能使用。windows推荐wamp。
> * 用JS单线程做这种高精度的定时，狗带。。
> * 作为过气音游狗，还是勉强写了个出来了。。
> * 键盘按键为DF和KL，按的时候渲染还是会出现卡顿。。
> * 食用时请下载到本地，自带好音乐文件和imd谱文件，置于index.html文件相同位置。
